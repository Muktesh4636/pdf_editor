🏦 Filter bank transactions

Upload your bank statement PDF, add one or more people (name can be partial), optional phone numbers, and we show only lines that match. Everything runs in your browser.

How matching works — for each person, a line is kept if the name text appears anywhere in that line (not case-sensitive), or the phone number appears (digits only, partial OK). You can add several people: a line is shown if it matches any of them. After filtering, lines are split into credit vs debit using common keywords (Cr/Dr, deposit, withdraw, UPI, etc.) and amounts are estimated from numbers on each line — verify totals with your bank. Scanned PDFs: we run OCR in your browser when there is no text layer (English; first run ~10–15 MB).
