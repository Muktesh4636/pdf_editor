🏦 Filter bank transactions

Upload one or more bank statement PDFs (combined into one search), add one or more people (name can be partial), optional phone numbers, and we show only lines that match. Everything runs in your browser.

How matching works — for each person, a line is kept if the name text appears anywhere in that line (not case-sensitive), or the phone number appears (digits only, partial OK). You can add several people: a line is shown if it matches any of them. Multiple PDFs are combined: every uploaded file is read and Find shows matching transactions from all of them (each hit shows which file it came from). Password PDFs: if a file is protected, a password box appears only then; we decrypt in your browser. You can download unlocked copies after they load. Amounts try to ignore long UPI/ID numbers; dates appear only if that PDF line actually contains a date. After filtering, credit/debit splits use keywords — verify with your bank. Scanned PDFs: OCR runs when there is no text layer (English; first run ~10–15 MB). OCR is capped at 45 pages total across all files in one batch.
